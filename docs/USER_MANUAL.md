# 누리나무 AI 법률통합지원 시스템 — 사용자 매뉴얼

**문서 번호**: NNA-UM-2026-001  
**버전**: v4.1  
**최종 수정일**: 2026년 4월  
**대상**: 시스템 최종 사용자 (공무원, 민원담당자)

---

## 1. 시스템 개요

누리나무 AI 법률통합지원 시스템은 행동강령, 청탁금지법, 공익신고 보호 등 공공 법규에 관한 질문에 AI가 실시간으로 응답하는 법률 통합지원 시스템입니다.

### 1.1 주요 기능

| 기능 | 설명 |
|------|------|
| **AI 법령 상담** | 자연어 질문으로 법령 및 지침 검색 |
| **법제처 법령 연동** | 최신 법령 실시간 참조 |
| **근거 출처 제공** | 답변의 참조 법령 목록 표시 |
| **대화 이력 관리** | 이전 상담 내용 조회 |
| **답변 내보내기** | PDF/텍스트 형식으로 저장 |

## 2. 시스템 접속

### 2.1 권장 브라우저

| 브라우저 | 버전 | 지원 여부 |
|---------|------|---------|
| Microsoft Edge | 최신 버전 | ✅ 권장 |
| Chrome | 최신 버전 | ✅ 지원 |
| Firefox | 최신 버전 | ✅ 지원 |
| Internet Explorer | 모든 버전 | ❌ 미지원 |

### 2.2 접속 방법

1. 인트라넷 웹 브라우저 실행
2. 시스템 URL 입력: `http://[기관내부주소]/`
3. 메인 화면 확인

## 3. 화면 구성

```
┌────┬──────────────────────────────────────┐
│    │  헤더 (시스템명 / 보안 상태)          │
│ ① │────────────────────────────────────── │
│    │                                      │
│ 사 │           ② 대화 영역                │
│ 이 │    (AI 답변 및 사용자 질문 표시)      │
│ 드 │                                      │
│ 바 │────────────────────────────────────── │
│    │     ③ 질문 입력 영역                  │

```

| 번호 | 영역 | 설명 |
|------|------|------|
| ① | 사이드바 | 이전 대화 목록, 새 대화 시작 |
| ② | 대화 영역 | AI 답변 및 내 질문 표시 |
| ③ | 입력 영역 | 질문 입력 및 전송 |

---

## 4. 질문하기

### 4.1 기본 사용법

1. 하단 입력창에 궁금한 내용을 한국어로 입력합니다
2. **Enter** 키 또는 **전송 버튼(↑)** 클릭
3. AI가 법령을 분석 후 답변을 생성합니다 (보통 3~15초 소요)
4. 답변 하단의 **참조 법령** 목록으로 출처를 확인합니다

### 4.2 효과적인 질문 방법

✅ **좋은 예시 (구체적)**
> "과장급 공무원이 민간 기업 대표로부터 10만원 상당의 골프 접대를 받았을 때 청탁금지법 위반 여부는?"

❌ **나쁜 예시 (모호한)**
> "접대 받아도 돼요?"

### 4.3 자주 묻는 질문 유형

- `청탁금지법상 [선물/식사/경조사비] 허용 기준`
- `행동강령 위반 시 처벌 및 신고 절차`
- `공익신고자 보호 대상 및 지원 내용`
- `이해충돌방지법 적용 범위`

### 5.2 답변 활용

| 버튼 | 기능 |
|------|------|
| 복사 아이콘 | 답변 전체 클립보드 복사 |
| 👍 (엄지 위) | 도움이 됐어요 (피드백) |
| 👎 (엄지 아래) | 개선이 필요해요 (피드백) |

> **⚠️ 주의**: AI 답변은 참고용입니다. 중요한 법적 판단은 반드시 관련 부서 또는 법무담당관에게 확인하세요.

## 6. 대화 이력 관리

- 좌측 사이드바에서 **이전 대화 목록** 확인 가능

## 7. 키보드 단축키 (접근성)

|--------|------|

| `Shift + Enter` | 줄 바꿈 |
